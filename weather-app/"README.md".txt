# 🌦️ Interactive Python Weather App

A simple but powerful Python app that fetches and displays current weather information based on user input. Uses OpenWeatherMap's API and provides a friendly interface.

---

## ✨ Features

- 🔍 Takes user input for a city and fetches real-time weather data
- 🌡️ Displays temperature, humidity, and weather conditions
- 🖥️ Interactive user experience via command line
- 🔌 Demonstrates basic API integration in Python

---

## 🧰 Tech Stack

- Python
- Requests
- OpenWeather API
